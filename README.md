# AI Smart Email Assistant (Java OOP + Swing + Security)

AI Smart Email Assistant is a desktop-based email management system developed in Java. The project demonstrates Object-Oriented Programming principles along with GUI development, authentication, and security features. It helps users manage emails efficiently through smart categorization, priority detection, auto-reply suggestions, and secure data handling.

---

## Features

- Secure Sign Up and Login System
- Password Hashing using SHA-256
- Email Management (Add and View Emails)
- Smart Email Categorization (Work, Spam, Finance, etc.)
- Priority Detection (Important and Normal Emails)
- Auto Reply Suggestion System
- Email Encryption and Decryption using Base64
- Java Swing-based Graphical User Interface
- Organized MVC-style Architecture (Model, Service, UI)

---
